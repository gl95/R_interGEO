workonData {interGEO}R Documentation

workonData

Description

Performs basic preliminary operations on data.

Usage

workonData(env0)

Details

Data columns (samples), numeric rows (probes), text rows (factors and levels) are visualized in four different panels. You can perform different operations related to each panel.

"SAMPLE SELECT" command calls sampleSelect function.

"PROBE SELECT" command calls geneSelect function.

"ADD FACTOR" command calls getFactor function.

"RENAME LEVEL" command calls renameLevel function.

"DELETE FACTOR" command calls deleteFactor function.

Eventually, it uploads data object to global environment.
